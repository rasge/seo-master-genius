# Libertad PRO Digital - High-Conversion Agency Landing Page

A premium, Neil Patel-inspired landing page built for high conversion, search engine dominance, and lead generation. This project features a modern, automated SEO methodology and a sleek, animated user interface.

## 🚀 Features

- **Neil Patel Style Hero**: Centralized lead generation input for URL analysis with sleek glassmorphism effects.
- **Interactive Methodology**: 4-step process visualization with a mock ROI dashboard.
- **Premium Design System**: Glassmorphism navbar, grid backgrounds, and custom floating animations.
- **Lead Generation**: Integrated CTA sections and input fields for capturing prospects.
- **Optimized for SEO**: Full metadata configuration, semantic HTML, and image optimization using `next/image`.
- **Performance Focused**: Built with Next.js 15, Turbopack, and Tailwind CSS 4.0.

## 🛠️ Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS 4.0 (Vanilla CSS first approach)
- **Icons**: Lucide React
- **Animations**: CSS Keyframes & Framer Motion (ready for integration)
- **Deployment**: Vercel ready

## 📦 Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the outcome.

## 📝 Project Structure

- `src/app/`: Core page routing and layout.
- `src/components/`: Modular UI sections (Hero, Process, Testimonials, etc.).
- `src/app/globals.css`: Design tokens, glassmorphism utilities, and custom animations.
